# restocking-app


### Requirement
```
 * "node": ">= 6.0.0",
 * "npm": ">= 3.0.0"

 ```

### Uses
```
* Vue.js, with Vuetify and Vuelidate
* Firebase

```

# install dependencies
```
npm install

```

# serve with hot reload at localhost:8080
```
npm run dev

```
